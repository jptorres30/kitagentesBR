Preciso saber programar para usar?
+
Não. O kit inclui onboarding interativo que guia do zero — instalação, configuração e primeiro uso. Se você sabe copiar e colar texto, consegue usar.
Quanto custa usar depois que comprar?
+
Usando o Antigravity, autenticação é via Google — sem custo extra. Via Claude Code standalone, precisa de créditos da Anthropic (a partir de $5 USD). Custo real: R$0,50 a R$2,50 por ciclo completo.
Funciona para qualquer nicho?
+
Sim. O kit foi construído com os big nichos BR — finanças, saúde, emagrecimento, relacionamento, espiritualidade, produtividade — e seus subnichos mais rentáveis. O agente calibra linguagem e visual automaticamente.
Qual a diferença entre mensal e vitalício?
+
Conteúdo idêntico. No vitalício você paga uma vez e tem acesso para sempre, incluindo atualizações futuras. No mensal paga R$24,90/mês e cancela quando quiser. Para quem usa com frequência, o vitalício se paga em 3 meses.
Como recebo o acesso após a compra?
+
Imediatamente após confirmação do pagamento você recebe o link da página de instruções com download do kit e passo a passo completo de instalação.
Posso usar para vender meus próprios produtos?
+
Sim, essa é a principal finalidade. Use os agentes para criar e lançar seus próprios infoprodutos mais rápido. Sem restrição de uso comercial.
